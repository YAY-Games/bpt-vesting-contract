# bpt-vesting-contract

- Language: Solidity v0.8.10

- Project framework: hardhat + truffle / web3

- Nodejs: v14.17.0


## Installation & Usage

1. Install packages
```
yarn
```

2. Build project
```
yarn build
```

### Testing

```
yarn test
```

### Run linter

```
yarn lint
```
